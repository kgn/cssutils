  5  This module has one importable object, cssvalues, that contains a  
  6  dictionary of compiled regular expressions.  The keys of this  
  7  dictionary are all of the valid CSS property names.  The values 
  8  are compiled regular expressions that can be used to validate 
  9  the values for that property.  (Actually, the values are references
