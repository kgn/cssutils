  6  Unknown properties 
  7  ------------------ 
  8  User agents must ignore a declaration with an unknown property. 
  9  For example, if the style sheet is:: 
 10   
 11      H1 { color: red; rotation: 70minutes } 
 12   
 13  the user agent will treat this as if the style sheet had been:: 
 14   
 15      H1 { color: red } 
 16   
 17  Cssutils gives a warning about an unknown CSS2 Property "rotation". 
 18   
 19  Illegal values 
 20  -------------- 
 21  User agents must ignore a declaration with an illegal value. For example:: 
 22   
 23      IMG { float: left }       /* correct CSS2 */ 
 24      IMG { float: left here }  /* "here" is not a value of 'float' */ 
 25      IMG { background: "red" } /* keywords cannot be quoted in CSS2 */ 
 26      IMG { border-width: 3 }   /* a unit must be specified for length values */ 
 27   
 28  A CSS2 parser would honor the first rule and ignore the rest, as if the 
 29  style sheet had been:: 
 30   
 31      IMG { float: left } 
 32      IMG { } 
 33      IMG { } 
 34      IMG { } 
 35   
 36  Cssutils again will issue warning about invalid CSS2 property values. 
 37
